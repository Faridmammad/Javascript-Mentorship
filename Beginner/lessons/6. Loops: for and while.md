## What is a loop?

A **loop** lets you run the same code multiple times without writing it over and over.

## 1. `for` loop

The most common loop. Used when you know **how many times** to repeat.

### Syntax:
```javascript
for (initialization; condition; increment) {
  // code to repeat
}
```  


### Example: Count from 1 to 5
```javascript
for (let i = 1; i <= 5; i++) {
  console.log("Count: " + i);
}
```  


ğŸ‘‰ Output:
```javascript
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
```  


### How it works:
- `let i = 1` â†’ start at 1
- `i <= 5` â†’ continue while i is less than or equal to 5
- `i++` â†’ increase i by 1 after each loop

## 2. `while` loop

Runs code **while** a condition is `true`. Use when you donâ€™t know how many times.

### Syntax:
```javascript
while (condition) {
  // code to repeat
  // must update condition inside
}
``` 


### Example: Count from 1 to 5
```javascript
let i = 1;
while (i <= 5) {
  console.log("Count: " + i);
  i++; // update i
}
``` 


> ğŸ”¹ If you forget `i++`, the loop runs forever! (infinite loop)

## Common uses of loops

### 1. Print numbers
```javascript

  console.log(i);
}
``` 


### 2. Print even numbers

```javascript

  sum += i; // sum = sum + i
``` 

### 4. Repeat a message
